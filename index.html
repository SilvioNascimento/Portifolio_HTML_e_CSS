Sobre Mim

Graduando em Ciência da Computação pela UFPB, atuo como Full Stack Developer focado em criar soluções eficientes desde a estruturação do banco de dados até a entrega em produção.

Possuo 8 meses de experiência prática através do projeto de extensão da Proex, onde fui responsável pelo desenvolvimento de um sistema completo de Gestão de Sementes e Viveiro de Mudas para a AFINK. Minha atuação envolveu desde a prototipagem do sistema pelo Figma, modelagem de diagrama de classe e criação de API e testes de suas rotas até a integração com o front-end e hospedagem do sistema, utilizando as seguintes tecnologias:

  • Linguagem de Programação: Java 21, JavaScript;
  • Back-end: Spring Boot (Framework);
  • Front-end: HTML5, CSS3, React.js;
  • Segurança e Autenticação: Spring Security, JWT;
  • Manipulação de dados (ORM): Hibernate;
  • Banco de Dados: PostgreSQL 16;
  • Container/Cloud Computing: Docker, Hostinger (Deploy/Hospedagem);
  • Versionamento: Git, GitHub;
  • Testes de rotas: Postman.

Sou movido por desafios técnicos e estou em busca da minha primeira oportunidade como desenvolvedor júnior para somar a uma equipe focada em inovação e qualidade técnica.
